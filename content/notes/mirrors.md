> **توسعه در شرایط انزوا:** با وجود محدودیت‌های جدی، جامعه توسعه‌دهندگان ایران همچنان به دنبال راه‌حل‌هایی برای تداوم کار خود است.

### مخازن پایتون (PyPI)

یکی از چالش‌های اصلی توسعه‌دهندگان پایتون در شرایط فعلی، دسترسی به مخزن PyPI است. برای نصب پکیج‌های پایتون می‌توانید از میرورهای زیر استفاده کنید:

| میرور | آدرس | توضیحات |
|-------|------|---------|
| رانفلر (Runflare) | **mirror-pypi.runflare.com/simple/** | میرور ایرانی قابل اعتماد با سرعت مناسب |
| سازمان فناوری اطلاعات | **archive.ito.gov.ir/python** | ممکن است پکیج‌های جدیدتر را نداشته باشد |
| جامکو | **pypi.jamko.ir/simple** | توسط سازمان فناوری اطلاعات ایران نگهداری می‌شود |

برای نصب پکیج با استفاده از میرور رانفلر، از دستور زیر استفاده کنید:
```bash
pip install --trusted-host mirror-pypi.runflare.com -i https://mirror-pypi.runflare.com/simple/ نام-پکیج
```
برای تنظیم دائمی این میرور در pip:
```bash
pip config --user set global.index https://mirror-pypi.runflare.com/simple
pip config --user set global.index-url https://mirror-pypi.runflare.com/simple
pip config --user set global.trusted-host mirror-pypi.runflare.com
```

## نکات حیاتی برای توسعه در شرایط اینترنت ملی

### ۱. آرشیو‌سازی و ذخیره‌سازی محلی
با توجه به غیرقابل پیش‌بینی بودن وضعیت دسترسی، **حتماً از منابع مهم آرشیو آفلاین تهیه کنید**. برای پکیج‌های پایتون:
```bash
# دانلود پکیج‌ها در پوشه محلی
pip download -r requirements.txt -d packages

# نصب آفلاین از پوشه محلی
pip install --no-index --find-links=packages -r requirements.txt
```

### ۲. شناسایی میرورهای قابل اعتماد
برخی میرورها ممکن است به دلیل وابستگی به اینترنت جهانی، در شرایط خاص کار نکنند. به عنوان مثال، **mirrors.kubarcloud.com** فقط در صورتی کار می‌کند که اینترنت دیتاسنترها ملی نشده باشد.

### ۳. حل مشکلات وابستگی (Dependency)
گاهیاوقات در نصب پکیج‌های با وابستگی‌های زیاد به مشکل می‌خورید. در این حالت، پکیج‌های مشکل‌ساز را جداگانه نصب کنید:
```bash
# اگر فلاسک (flask) به مشکل برخورد
pip install markupsafe  # نصب وابستگی مشکل‌ساز
pip install flask      # سپس نصب پکیج اصلی
```

### ۴. به‌روزرسانی مستندات
این فهرست بر اساس وضعیت ۲۵ دی ۱۴۰۴ (۲۵ ژانویه ۲۰۲۶) گردآوری شده است. ممکن است برخی سرویس‌ها در آینده در دسترس نباشند یا وضعیت آنها تغییر کند. پیشنهاد می‌کنیم برای اطلاعات به‌روز، با جامعه توسعه‌دهندگان ایرانی در ارتباط باشید.

### ۵. امنیت و حریم خصوصی
در استفاده از سرویس‌های داخلی، به‌ویژه پیام‌رسان‌ها و پلتفرم‌های هوش مصنوعی، مسائل امنیتی و حریم خصوصی را در نظر بگیرید. برخی کاربران به دلیل نگرانی‌های حریم خصوصی از استفاده از برخی اپلیکیشن‌های داخلی خودداری می‌کنند.

## سخن پایانی

اگرچه شرایط کنونی برای جامعه فناوری ایران چالش‌برانگیز است، اما وجود این منابع نشان می‌دهد که می‌توان با تطبیق و خلاقیت به فعالیت ادامه داد. پیشنهاد می‌کنیم در صورت دسترسی مجدد به اینترنت جهانی، حتماً از مهم‌ترین منابع مورد نیاز خود آرشیو آفلاین تهیه کنید.

**تجربیات و منابع دیگری دارید که در این فهرست موجود نیست؟** لطفاً آنها را با سایرین به اشتراک بگذارید تا این راهنما تکمیل‌تر شود. به یاد داشته باشید که همکاری و اشتراک‌گذاری دانش در شرایط سخت، کلید بقا و پیشرفت است.
